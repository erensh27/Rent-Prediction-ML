# House Rent Prediction System

## Objective
To predict monthly house rent based on city and property features using machine learning.

## Dataset
House Rent Prediction Dataset (India)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Random Forest Regressor
- Flask
- HTML/CSS

## Methodology
1. Data preprocessing and cleaning
2. Feature encoding and selection
3. Model training using Random Forest
4. Model persistence using joblib
5. Web deployment using Flask

## Output
The system predicts house rent through a web interface based on user input.

## Conclusion
The project demonstrates an end-to-end machine learning pipeline with real-time prediction capability.
